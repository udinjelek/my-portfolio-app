About Me

In 2010, I started my career in the telecommunications industry as a 'Tools Maker,' creating programs to collect data, generate reports, and develop automation tools. In 2022, I transitioned to a full-stack developer role at different Company, where I build dynamic websites to support the HR team.

My focus now is on crafting accessible and efficient software that balances design and engineering.

When I'm not working, I enjoy spending time with my family, drawing, and enhancing my logic programming skills by making games, which often require complex logic and creativity.

Personal Project

Customer Chat App

n Care Chat is a super handy chat app I built for customer care purpose. It works great on both laptops and phones, so you can chat with support anytime, anywhere. The app is responsive and easy to use, making customer service fast and efficient. Plus, you can attach images and files, making it even more versatile

Care Chat utilizes Angular for the frontend, Flask for the backend, and websocket technology for real-time communication. SQLite serves as the database, ensuring efficient data management for fast and versatile customer support.

Telegram File Converter Bot

The Telegram File Converter Bot is a practical tool I developed to simplify the conversion of HTML data tables into readable CSV files. Designed for ease of use, this bot operates within Telegram to prevent the need for additional software installation. It supports seamless data handling, making it perfect for database imports and Excel tasks like VLOOKUP.

Powered by Python, the bot ensures efficient and accurate data conversion while keeping track of user interactions.

Greeting Cards App

The Greeting Cards App is a creative solution I developed to make sending holiday messages more enjoyable and personalized. Instead of plain text messages, this app allows me to choose from various templates tailored that I made before for different recipients, whether it's my boss, colleagues, friends, or family. Each template can be customized with the recipient's name through a simple URL modification, ensuring a personal touch while keeping the process quick and easy.

All the artwork is vector-based and created by me with Inkscape. The programming is done using Angular with TypeScript.

Puzzle Connection Game

Sometimes I need to hone my programming skills and satisfy my desire to solve logic challenges. That's why I created Game. Making games often requires complex logic, involving various algorithms and mathematical calculations that are rarely encountered in full-stack development. This game serves as a fun and engaging way to improve those skills.

The Game features a captivating narrative with character dialogues that lead to a choice-driven ending.

Inspired by Onet, The game throws in some twists and additional mechanics that demand strategic thinking to solve the puzzles.

The drawings are raster-based, created entirely by me in Krita, add a unique visual style. The game engine used is Godot, with GDScript as the language of choice.
